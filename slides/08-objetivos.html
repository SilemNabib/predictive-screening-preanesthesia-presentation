03 · Objetivos y metodología

Objetivos del proyecto

Objetivo general

Diseñar, entrenar y validar un modelo predictivo de clasificación para la identificación de pacientes que requieren valoración preanestésica formal, utilizando como proxy la ocurrencia de eventos clínicos posoperatorios relevantes, a partir de información clínica preoperatoria estructurada, y garantizando trazabilidad técnica para su integración.

OE1

Datos y gobierno

Dataset anonimizado, documentado con diccionario, manejo de faltantes y trazabilidad reproducible.

OE2

Desarrollo

Feature set diseñado, 9 algoritmos entrenados con validación interna, código y versiones almacenadas.

OE3

Evaluación

Métricas clínicas y estadísticas, calibración, análisis de error y subgrupos sobre datos de prueba.

OE4

Servicio (API)

API REST que consume el modelo, entrega recomendaciones y registra interacciones para auditoría.

OE5

Integración

Flujo de implementación e integración del modelo con los sistemas institucionales de la FVL.
